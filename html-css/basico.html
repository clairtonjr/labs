HTML5

HTML é uma linguagem de marcação que estrutura uma pagina web. Têm uma série de elementos marcadores diferentes para que o conteúdo seja ou apareça de uma determinada maneira. Essa página está sendo exibida por HTML, apenas com a estruta básica de marcadores como html, head, title e body. Para dar estrutura e significado a um texto seja ele de um jornal, revista ou livro usamos de títulos, parágrafos, listas, ênfases, citações e etc. O HTML têm tags (marcações) para tornar estruturado e legível o texto exibido no navegador.

Títulos e Subtítulos

Existem seis elementos para títulos: H1, H2, H3, H4, H5 e H6. A importância é H1 > H2, H2 > H3 e assim sucessivamente.

Dicas de uso: preferencialmente usar não mais que três tags de títulos e apenas um H1 por página.

Listas

Listas fazem parte de muitos momentos da nossa vida, também de boa parte da web. No HTML temos listas ordenadas e não ordenadas.

- Lista não ordenada - a posição de cada elemento não tẽm importância.

  • Sal
  • Limão
  • Leite

- Lista ordenada - a posição de cada elemento têm importância.

  1. Entrar na fila
  2. Pagar estacionamento
  3. Ir embora

Tabelas

Tabelas são um conjunto estuturado de dados constituído por linhas e colunas. uma tabela permite consultar permite consultar de forma fácil e rápida valores.

Em versões anteriores do HTML e em muitos sites antigos, usavamos tabelas para o layout do site, essa já se tornou uma má prática. Sabemos o que são tabelas, em HTML usamos para mostrar dados dados tabulares. Além de não usarmos para fazer layout, devemos evitor o uso porquê elas não são automaticamente responsivas.
